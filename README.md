PHPAV
=====

PHP CLI Virus/Malware Scanner

A simple console framework for detected infected PHP files among Web servers.

@copyright [Nexylan.com](http://www.nexylan.com) - 2016

Usage
---
### via CLI

```
php scan.php /path/to/scan
```
